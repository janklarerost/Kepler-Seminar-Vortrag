# 3. Keplersches Gesetz mittels komplexer Integration

Dieses Repository enthält die LaTeX-Quellen und die fertige PDF-Präsentation für den Vortrag

## Inhalt
- Herleitung des 3. Keplerschen Gesetzes für elliptische Bahnen.
- Fokus auf die Anwendung des **Cauchy-Integralsatzes** und der komplexen Integration zur Lösung des reell "unlösbaren" T-Integrals.
